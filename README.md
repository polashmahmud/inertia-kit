# Inertia Kit

Vue UI scaffolding for Inertia.js projects. Install the CLI, pick a component, and drop the generated Vue files straight into your app.

## Install

```bash
npm i @polashmahmud/inertia-kit
```

## Use

- List all available component templates with `npx inertia-kit list`.
- Scaffold a component into your project with `npx inertia-kit add Input` (use `--dest` to choose a folder).
- Edit the generated `.vue` file to match your form or layout.
